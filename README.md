# 🤖 AI Code Reviewer

An AI-powered code review tool that analyzes your code and provides professional feedback on bugs, improvements, and best practices.

## 🚀 Live Demo
Run locally on `http://localhost:3000`

## ✨ Features
- Paste any code and get instant AI feedback
- Identifies bugs and security issues
- Suggests improvements and best practices
- Supports all programming languages
- Powered by multiple AI models via OpenRouter

## 🛠️ Tech Stack
- Node.js
- Express.js
- OpenRouter API
- JavaScript
- HTML/CSS

## ⚙️ How to Run

1. Clone the repo
```bash
git clone https://github.com/sami-gujjar/ai-code-reviewer.git
cd ai-code-reviewer
```

2. Install dependencies
```bash
npm install
```

3. Create `.env` file

4. 4. Run the app
```bash
node index.js
```

5. Open browser at `http://localhost:3000`

## 📸 How It Works
1. Paste your code in the text area
2. Click "Review My Code"
3. AI analyzes and returns detailed feedback
4. Fix your code based on suggestions

## 🤖 AI Models Used
- OpenRouter Free Tier
- Multiple AI models for best results
